# p3-001 — Scaffold dashboard-app-1 React+Vite app

**Status:** `pending`

Scaffold a React + Vite app under `ideations/dashboard-app-1/` using the Nx React plugin.

```bash
pnpm nx g @nx/react:app dashboard-app-1 --directory=ideations/dashboard-app-1 --bundler=vite --style=css --routing=false --e2eTestRunner=none
```

**Definition of done:**
- `ideations/dashboard-app-1/` directory exists with standard Nx React+Vite structure
- `pnpm nx dev dashboard-app-1` starts a dev server (default port, to be changed in p3-002)
- `pnpm nx build dashboard-app-1` produces a static bundle
