# p3-002 — Configure Vite and set up Tailwind v4 + shadcn deps

**Status:** `pending`

Configure the scaffolded app for the shell's asset serving requirements and install the UI dependencies needed for shadcn/ui with Tailwind CSS v4.

## Vite configuration (`ideations/dashboard-app-1/vite.config.mts`)

- `base: '/apps/dashboard-app-1/'` — required so built asset paths resolve correctly when served by the shell
- `server.port: 4203` / `preview.port: 4203`
- `cacheDir: '../../node_modules/.vite/ideations/dashboard-app-1'`
- Add `@tailwindcss/vite` to the plugins array

## TypeScript path alias

Add `@/*` → `./src/*` path alias to `vite.config.mts` and `tsconfig.app.json` (required by shadcn component imports).

## Dependencies (`ideations/dashboard-app-1/package.json`)

Add:
```json
{
  "dependencies": {
    "class-variance-authority": "*",
    "clsx": "*",
    "tailwind-merge": "*",
    "lucide-react": "*",
    "@radix-ui/react-collapsible": "*",
    "@radix-ui/react-separator": "*",
    "@radix-ui/react-avatar": "*",
    "@radix-ui/react-dropdown-menu": "*",
    "@radix-ui/react-slot": "*",
    "@radix-ui/react-tooltip": "*"
  },
  "devDependencies": {
    "tailwindcss": "*",
    "@tailwindcss/vite": "*"
  }
}
```

Run `pnpm install` from workspace root to link new deps.

## Tailwind + CSS variables (`ideations/dashboard-app-1/src/styles.css`)

Replace generated content with:
- `@import "tailwindcss";`
- Full set of CSS custom properties for `new-york-v4` zinc color palette (light + dark mode via `.dark` class)
- `--radius` variable

## `components.json` (`ideations/dashboard-app-1/components.json`)

Create for shadcn CLI:
```json
{
  "$schema": "https://ui.shadcn.com/schema.json",
  "style": "new-york-v4",
  "rsc": false,
  "tsx": true,
  "tailwind": {
    "css": "src/styles.css",
    "baseColor": "zinc",
    "cssVariables": true
  },
  "aliases": {
    "components": "@/components",
    "utils": "@/lib/utils",
    "ui": "@/components/ui",
    "lib": "@/lib",
    "hooks": "@/hooks"
  }
}
```

## `src/lib/utils.ts`

Create with the `cn()` helper:
```ts
import { clsx, type ClassValue } from 'clsx';
import { twMerge } from 'tailwind-merge';

export function cn(...inputs: ClassValue[]): string {
  return twMerge(clsx(inputs));
}
```

**Definition of done:**
- `pnpm nx dev dashboard-app-1` serves on port 4203
- Tailwind styles apply to the app
- `@/` imports resolve correctly
