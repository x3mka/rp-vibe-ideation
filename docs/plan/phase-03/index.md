**Status:** `done`

## Goal

Build real ideation sub-apps that demonstrate meaningful UI patterns. Each sub-app follows the established Phase 02 pattern: scaffolded under `ideations/<app-name>/`, built with a base path of `/apps/<app-name>/`, copied into the shell's `public/apps/` folder, and registered in the ideation registry.

Tasks are numbered `P3-001`, `P3-002`, etc.

## Sub-Apps

### dashboard-app-1
