# p3-005 — Wire build infrastructure and register app

**Status:** `pending`

Connect `dashboard-app-1` into the Nx build graph and the ideation registry so the shell can serve it.

## 1. Copy target (`ideations/dashboard-app-1/project.json`)

Ensure the `copy` target is present (may need to add it if the scaffold didn't create it):

```json
{
  "name": "@rp-vibe-ideation/dashboard-app-1",
  "targets": {
    "copy": {
      "dependsOn": ["build"],
      "command": "node -e \"require('fs').cpSync('ideations/dashboard-app-1/dist', 'apps/web/public/apps/dashboard-app-1', {recursive:true,force:true})\"",
      "options": { "cwd": "{workspaceRoot}" }
    }
  }
}
```

## 2. Shell build dependency (`apps/web/project.json`)

Add `"dashboard-app-1:copy"` to both `build.dependsOn` and `dev.dependsOn`:

```json
{
  "targets": {
    "build": {
      "dependsOn": ["react-app:copy", "angular-app:copy", "dashboard-app-1:copy"]
    },
    "dev": {
      "dependsOn": ["react-app:copy", "angular-app:copy", "dashboard-app-1:copy"]
    }
  }
}
```

## 3. Registry entry (`packages/ideation-registry/src/lib/ideation-registry.ts`)

Add:
```ts
{
  id: 'dashboard-app-1',
  name: 'Dashboard App 1',
  description: 'shadcn sidebar-07 collapsible sidebar dashboard',
  group: 'Dashboards',
  url: '/apps/dashboard-app-1',
  devUrl: 'http://localhost:4203',
},
```

**Definition of done:**
- `pnpm nx build web` completes without errors (builds all sub-apps, copies assets, builds shell)
- `apps/web/public/apps/dashboard-app-1/` is populated after the build
- The app appears in the shell's app-switcher under the "Dashboards" group
- Shell route `/ideation/dashboard-app-1` loads the app in an iframe
