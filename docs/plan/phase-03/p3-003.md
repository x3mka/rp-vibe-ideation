# p3-003 — Add shadcn UI components

**Status:** `pending`

Use the shadcn CLI to add the component primitives needed for the sidebar-07 layout. Run from the `ideations/dashboard-app-1/` directory so the CLI picks up `components.json`.

```bash
cd ideations/dashboard-app-1
pnpm dlx shadcn@latest add sidebar collapsible breadcrumb avatar dropdown-menu separator
```

This generates files under `src/components/ui/`:
- `sidebar.tsx` — full SidebarProvider, Sidebar, SidebarHeader/Content/Footer, SidebarRail, SidebarInset, SidebarTrigger, SidebarGroup, SidebarMenu, SidebarMenuItem, SidebarMenuButton, SidebarMenuAction, SidebarMenuSub/SubItem/SubButton, useSidebar hook
- `collapsible.tsx` — Collapsible, CollapsibleContent, CollapsibleTrigger
- `breadcrumb.tsx` — Breadcrumb, BreadcrumbList, BreadcrumbItem, BreadcrumbLink, BreadcrumbPage, BreadcrumbSeparator
- `avatar.tsx` — Avatar, AvatarImage, AvatarFallback
- `dropdown-menu.tsx` — DropdownMenu and all sub-components
- `separator.tsx` — Separator

Also adds any peer components the CLI pulls in automatically (e.g. `tooltip`, `sheet` for mobile sidebar).

**Definition of done:**
- All component files exist under `src/components/ui/`
- No TypeScript errors in the generated files (`pnpm nx typecheck dashboard-app-1` or `tsc --noEmit`)
