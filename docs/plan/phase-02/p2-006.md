# p2-006 — Use shadcn Tooltip on "Open in new tab" button

**Status:** `done`

Replace the native `title` attribute tooltip on the "Open in new tab" button in `ShellHeader` with the shadcn/ui `Tooltip` component.

```bash
pnpm dlx shadcn@latest add tooltip
```

Wrap the button:
```tsx
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger asChild>
      <button ...>
        <ExternalLink ... />
      </button>
    </TooltipTrigger>
    <TooltipContent>Open in new tab</TooltipContent>
  </Tooltip>
</TooltipProvider>
```

**Files:**
- `apps/web/src/components/ui/tooltip.tsx` — generated by shadcn
- `apps/web/src/components/ShellHeader.tsx`

**Definition of done:**
- Hovering the icon shows the shadcn tooltip (not the native browser tooltip)
- `title` attribute removed from the button
- `pnpm nx test web` passes
- `pnpm nx lint web` passes
