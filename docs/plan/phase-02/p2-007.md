# p2-007 — Replace native select with shadcn Combobox for app switcher

**Status:** `done`

Replace the native `<select>` in `ShellHeader` with a shadcn Combobox (Command + Popover) rendered by `AppSwitcher`. Groups are shown as `CommandGroup` headings; a `CommandSeparator` appears after each group except the last. Selecting an app or clearing the selection navigates as before.

```bash
pnpm dlx shadcn@latest add command popover button
```

`AppSwitcher` receives `activeId` and `onSelect` as props; `ShellHeader` owns routing and passes them down.

**Files:**
- `apps/web/src/components/ui/command.tsx` — generated
- `apps/web/src/components/ui/popover.tsx` — generated
- `apps/web/src/components/ui/button.tsx` — generated
- `apps/web/src/components/AppSwitcher.tsx` — rewritten as combobox
- `apps/web/src/components/ShellHeader.tsx` — use `<AppSwitcher>`; remove inline select

**Definition of done:**
- Clicking the trigger opens a popover with grouped app list and separators between groups
- Selecting an app navigates to `/ideation/<id>` and closes the popover
- Active app is visually indicated (checkmark)
- `pnpm nx test web` passes
- `pnpm nx lint web` passes
