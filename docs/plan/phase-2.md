# Phase 2 — React and Angular Example Apps

**Goal:** Two minimal co-located sub-apps (one React/Vite, one Angular) scaffolded in the monorepo, built with correct base paths, copied into the shell's `public/apps/` folder, and served by the shell as a single deployment artifact.

**Prerequisite:** Phase 1 complete.

---

## Tasks

---

### P2-001 — Create blank React sub-app

**Status:** `done`

Scaffold a minimal React + Vite app under `ideations/react-app/` using the Nx React plugin.

```bash
pnpm nx g @nx/react:app react-app --directory=ideations/react-app --bundler=vite --style=css --routing=false --e2eTestRunner=none
```

Adjust the generated app:
- Set the dev server port to **4201** in the Vite config
- Set the asset base path to `/apps/react-app/` in `vite.config.ts`:
  ```ts
  export default defineConfig({ base: '/apps/react-app/', ... })
  ```
- Replace generated content with a minimal blank page (app name + "React" label)
- Add a `copy` Nx target to `project.json`:
  ```json
  {
    "copy": {
      "dependsOn": ["build"],
      "command": "cp -r dist/ideations/react-app/. apps/web/public/apps/react-app/",
      "options": { "cwd": "{workspaceRoot}" }
    }
  }
  ```
- Register in the registry:
  ```ts
  { id: 'react-app', name: 'React App', description: 'Blank React + Vite example', group: 'Examples', url: '/apps/react-app', devUrl: 'http://localhost:4201' }
  ```

**Files:**
- `ideations/react-app/` — generated by Nx
- `ideations/react-app/vite.config.ts` — base path
- `ideations/react-app/project.json` — copy target
- `packages/ideation-registry/src/lib/ideation-registry.ts`

**Definition of done:**
- `pnpm nx dev react-app` starts a dev server at `http://localhost:4201`
- `pnpm nx build react-app` produces a static bundle with assets referencing `/apps/react-app/`
- `pnpm nx copy react-app` copies built assets to `apps/web/public/apps/react-app/`
- Shell loads the app from `/apps/react-app` in production
- Shell loads the app from `http://localhost:4201` in dev (when dev server is running)
- A smoke test asserts the root component renders — `pnpm nx test react-app` passes

### P2-002 — Create blank Angular sub-app

**Status:** `done`

Scaffold a minimal Angular app under `ideations/angular-app/` using the Nx Angular plugin.

```bash
pnpm nx g @nx/angular:app angular-app --directory=ideations/angular-app --style=css --routing=false --e2eTestRunner=none
```

Adjust the generated app:
- Set the dev server port to **4202** in the Angular config
- Set the asset base path to `/apps/angular-app/` in `angular.json`:
  ```json
  { "baseHref": "/apps/angular-app/" }
  ```
- Replace generated content with a minimal blank page (app name + "Angular" label)
- Add a `copy` Nx target to `project.json`:
  ```json
  {
    "copy": {
      "dependsOn": ["build"],
      "command": "cp -r dist/ideations/angular-app/browser/. apps/web/public/apps/angular-app/",
      "options": { "cwd": "{workspaceRoot}" }
    }
  }
  ```
- Register in the registry:
  ```ts
  { id: 'angular-app', name: 'Angular App', description: 'Blank Angular example', group: 'Examples', url: '/apps/angular-app', devUrl: 'http://localhost:4202' }
  ```

**Files:**
- `ideations/angular-app/` — generated by Nx
- `ideations/angular-app/project.json` — copy target + base href
- `packages/ideation-registry/src/lib/ideation-registry.ts`

**Definition of done:**
- `pnpm nx dev angular-app` (or `serve`) starts a dev server at `http://localhost:4202`
- `pnpm nx build angular-app` produces a static bundle with assets referencing `/apps/angular-app/`
- `pnpm nx copy angular-app` copies built assets to `apps/web/public/apps/angular-app/`
- Shell loads the app from `/apps/angular-app` in production
- Shell loads the app from `http://localhost:4202` in dev (when dev server is running)
- A smoke test asserts the root component creates successfully — `pnpm nx test angular-app` passes

### P2-003 — Wire shell build dependencies

**Status:** `done`

Configure the shell's `build` target to depend on both sub-app `copy` targets so that `pnpm nx build web` orchestrates the entire build in the correct order.

Add to `apps/web/project.json` (or `nx.json` targetDefaults if preferred):

```json
{
  "targets": {
    "build": {
      "dependsOn": ["react-app:copy", "angular-app:copy"]
    }
  }
}
```

**Files:**
- `apps/web/project.json` — or `nx.json`

**Definition of done:**
- `pnpm nx build web` alone builds both sub-apps, copies their assets, then builds the shell
- `apps/web/public/apps/react-app/` and `apps/web/public/apps/angular-app/` are populated before the Next.js build runs

### P2-004 — End-to-end smoke test

**Status:** `new`

Verify both sub-apps load correctly in production mode and dev mode.

**Production:**
1. `pnpm nx build web` — confirm full build succeeds (sub-apps built, copied, shell built)
2. `pnpm nx start web` — start the production shell
3. Open `http://localhost:3000`, confirm app switcher shows React App and Angular App
4. Select React App — confirm iframe loads from `/apps/react-app`
5. Select Angular App — confirm iframe loads from `/apps/angular-app`
6. Click "Open in new tab" for each — confirm both open standalone

**Development (HMR):**
7. `pnpm nx dev react-app` — confirm React app runs at `http://localhost:4201`
8. `pnpm nx dev angular-app` — confirm Angular app runs at `http://localhost:4202`
9. `pnpm nx dev web` — confirm shell starts at `http://localhost:3000`
10. Select React App — confirm iframe loads from `http://localhost:4201` (devUrl)
11. Select Angular App — confirm iframe loads from `http://localhost:4202` (devUrl)

**Definition of done:**
- All 11 steps pass without errors
- No TypeScript errors (`pnpm nx run-many -t typecheck`)
- No lint errors (`pnpm nx lint web`)
- All tests pass (`pnpm nx run-many -t test`)
